# Sales-dashboard
Get insights for your sales data in neat charts by just uploading the csv.

You can select your desired currency, you can generate bar chart for a minimum of any 2 columns you select.

You can check sales by category, month, payment methods and many more, any type of relevant csv file can be uploaded under 200mbs.

Usage:
First install streamlit and pandas:
pip install streamlit pandas

Then to run this program:
python -m streamlit run app.py

Upload CSV, select columns, pick currency, click Generate Dashboard

The dataset used to test this is the largest dataset of e-commerce sale in pakistan. However, you can use any other dataset you want to.

https://www.kaggle.com/datasets/zusmani/pakistans-largest-ecommerce-dataset
